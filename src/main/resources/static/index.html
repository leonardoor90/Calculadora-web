Construindo uma Calculadora Web com Java e Spring Boot

Este projeto demonstra a criação de uma calculadora interativa baseada na web, combinando um frontend intuitivo com um backend robusto em Java. A interface da calculadora é desenvolvida utilizando **HTML** e **CSS** para a estrutura e o estilo visual dos botões e do display. A lógica de interação inicial, como a captura dos números e operações digitadas, é gerenciada por **JavaScript** no navegador do usuário. No entanto, o verdadeiro poder de processamento reside no backend Java.

A comunicação entre o navegador e o servidor é estabelecida através de um **endpoint REST** implementado com **Spring Boot**. Quando o usuário clica no botão de igual (`=`), a expressão matemática é enviada do frontend para o backend Java via uma requisição HTTP **POST**. O servidor Spring Boot recebe essa expressão, realiza o cálculo utilizando lógica Java pura e, então, envia o resultado de volta para o navegador, que o exibe na tela da calculadora. Essa arquitetura separa claramente a apresentação (frontend) da lógica de negócios (backend), permitindo que cada parte seja desenvolvida e escalada de forma independente.
